## 概要
musiclyricsは、音楽の感想や歌詞への想いを投稿し、画像付きで一覧・閲覧できるWebアプリケーション。ユーザーは、好きな音楽、歌詞、感想、写真を投稿し、単に曲を紹介するだけでなく、ユーザー同士がコメントを通して感想や考察を共有できる点を重視し、音楽を通じて人とつながり、分かち合える体験を提供することを目指しました。

## サイトテーマ
「音歌詞やメロディの感じ方を共有し、新しい音楽の見方に出会う」

## テーマを選んだ理由
最近は、アーティストや流行をきっかけに音楽を聴く人が増える一方で、「この曲のどこが好きなのか」「なぜこの歌詞が刺さるのか」といった気持ちを言葉で共有できる場は多くないと感じていました。
そこで本アプリでは、好きな曲を投稿するだけでなく、楽曲の制作背景や歌詞の意味についてアーティストがコメントしたり、ユーザー同士が考察を共有できる場を提供します。
音楽を“聴くだけ”で終わらせず、言葉を通して分かち合うことで、より深く音楽を楽しみ、価値観の近い人と出会える場にしたいと考えています。
